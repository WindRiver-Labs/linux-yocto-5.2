* NXP S32GEN1 Reboot

Required properties:
- compatible :
  - "fsl,s32gen1-reset" for device that has an S32GEN1 compatible reboot
  mechanism
  - "fsl,s32gen1-mc_rgm" for device that supports functional reset escalation
- reg : Address and length of the register set for the device

Example:
mc_me: mc_me@40088000 {
	compatible = "fsl,s32gen1-reset";
	reg = <0x0 0x40088000 0x0 0x1000>;
};

mc_rgm: mc_rgm@40078000 {
	compatible = "fsl,s32gen1-mc_rgm";
	reg = <0x0 0x40078000 0x0 0x3000>;
};
